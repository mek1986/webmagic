TriangleFlagOptions

此类用来表示地图上构造一个三角旗的覆盖物类。

属性

属性 类型 默认值 说明
color string "#0000FF" 三角旗边线颜色。
weight number 3 三角旗边线的宽度，以像素为单位。
opacity number 0.5 三角旗边线的透明度（范围0-1 之间）。
fillColor String "#0000FF" 三角旗填充颜色。当参数为空时，覆盖物将没有填充效果。
fillOpacity number 0.2 三角旗填充的透明度（范围0-1 之间）。
lineStyle string "solid" 三角旗边线的样式（solid或dashed）。
