LayerGroup

此类是表示一个图层容器，用于存储T.Overlay的继承的图层。

构造函数

构造函数 说明
LayerGroup(layers:Array<Overlay>) 构建一个存储图层的容器。

方法

方法 类型 说明
addLayer(layer) None 从容器中增加一个图层。
removeLayer(layer) None 从容器中移除一个图层。
hasLayer(layer) Bool 判断图层是否在容器中。
getLayers() Array<Overlay> 得到容器中的所有图层。
clearLayers() None 清除容器中的所有图层。
setZIndex() None 设置容器Z值。
getLayer(id) Overlay 通过ID获取其对应的图层。
getLayerId(layer) String 获取容器中存在图层的唯一ID
