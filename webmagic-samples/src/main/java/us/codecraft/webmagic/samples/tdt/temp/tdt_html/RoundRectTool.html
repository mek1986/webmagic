RoundRectTool

构建一个绘制圆角矩形的工具类。

构造函数

构造函数 说明
RoundRectTool(map:Map,options:RoundRectToolOptions) 构造一个绘制圆角矩形的工具。

方法

方法 返回值 说明
clear() none 清空工具绘制所有圆角矩形。
getLayers() Array<RoundRect> 获取所有编辑完成的圆角矩形。
open() boolean 开启圆角矩形工具。
close() none 关闭圆角矩形工具。
addEventListener(event:String, handler:Function) none 添加事件监听函数。
removeEventListener(event:String, handler:Function) none 移除事件监听函数。

事件

事件 参数 说明
click click{type,target,currentLnglats,currentLayer,allLayers} 工具点击一次地图触发的事件。
move {type,target,currentLnglats,currentLayer,allLayers} 工具在绘制图形时移动鼠标的事件。
