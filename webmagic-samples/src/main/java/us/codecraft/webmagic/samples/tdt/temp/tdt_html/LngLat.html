LngLat

此类以经度和纬度表示的地理位置坐标点。

构造函数

构造函数 说明
LngLat (lng:Numbe,lat:Number) 根据给定经度和纬度创建地理位置坐标点。
参数说明：
lng：地理经度。
lat：地理纬度。

方法

方法 返回值 说明
getLng() number 获取地理坐标点的经度。
getLat() number 获取地理坐标点的纬度。
distanceTo(other:LngLat) number 计算当前地理坐标点与给定坐标点之间的距离。
参数说明：
other：经纬度坐标。
equals(other:LngLat) boolean 判断坐标点是否相等，当且仅当两点的经度和纬度均相等时返回true。
参数说明：
other：地理坐标。
