RoutePlan

此类表示一条驾车出行方案。没有构造函数，通过DrivingRouteResult.getPlan()方法获得。

方法

方法 返回值 说明
getNumRoutes() number 返回该方案包含的线路的个数。
getRoute(i:Number) Route 返回方案中索引指定的线路信息。索引0表示第一条线路。
getDistance() number 返回方案总距离，单位为公里。
getDuration() number 返回方案总时间，单位为秒。
getWayPois() Array<LngLat> 返回途径点。
getPath() Array<LngLat> 返回路线的经纬度。
