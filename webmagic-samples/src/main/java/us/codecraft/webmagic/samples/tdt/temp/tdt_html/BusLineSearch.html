BusLineSearch

公交路线搜索类。

构造函数

构造函数 说明
BusLineSearch(map:Map ,opts:BusLineSearchOptions) 公交路线搜索类。
参数说明：
map：地图对象。
opts：详见BusLineSearchOptions类。

方法

方法 返回值 说明
getBusList(keyword:String) none 检索公交信息。
getBusLine(busLstItem:BusListItem) none 检索公交路线。
setGetBusListCompleteCallback(callback:Function) none 设置公交列表查询后的回调函数。
setGetBusLineCompleteCallback(callback:Function) none 设置公交线路查询后的回调函数。
