LocalSearch

此类用于位置检索、周边检索和范围检索。

构造函数

构造函数 说明
LocalSearch(map:Map ,opts:LocalSearchOptions) 创建一个搜索类实例。
参数说明：
map：地图对象。
opts：详见LocalSearchOptions类。

方法

方法 返回值 说明
search(keyword:String,type:Number) none 根据检索词发起检索。
参数说明：
keyword：关键字。
type：搜索类型,1表示普通搜索;2表示视野内搜索;4表示普通建议词搜索;5表示公交规划建议词搜索;7表示 纯地名搜索(不搜公交线）;10表示拉框搜索
searchInBounds(keyword:String, bounds:Bounds) none 根据范围和检索词发起范围检索。
keyword：关键字。
bounds：搜索范围
searchNearby(keyword:String,center:lngLat, radius:Number) none 根据中心点、半径与检索词发起周边检索。
参数说明：
keyword：关键字。
center：中心点。。
Radius：搜索半径
getResults() LocalSearchResult 返回最近一次检索的结果。
clearResults() none 清除最近一次检索的结果。
gotoPage(page:Number) none 检索特定页面的结果。
参数说明：
page：检索特定页面数值。
setSpecifyAdminCode(code:Number) none 设置检索范围。
参数说明：
code：行政区的国标码。
setQueryType(type:Number) none 设置搜索类型。
参数说明：
type：搜索类型,1表示普通搜索;2表示视野内搜索;4表示普通建议词搜索;5表示公交规划建议词搜索;7表示 纯地名搜索(不搜公交线）。
getQueryType() number 返回搜索类型。
setPageCapacity(count:Number) none 设置每页容量。
参数说明：
count：每页容量。
getPageCapacity() number 返回每页容量。
setSearchCompleteCallback(fun:Function) none 设置检索结束后的回调函数。
参数说明：
fun：回调函数。回调函数参数为localSearchResult，具体内容详见localSearchResult类。
firstPage() none 检索第一页。
nextPage() none 检索下一页
previousPage() number 检索上一页。
lastPage() none 检索最后一页。
getCountNumber() number 返回总记录数。
getCountPage() none 返回共分总页数。
getPageIndex() string 返回当前页。
