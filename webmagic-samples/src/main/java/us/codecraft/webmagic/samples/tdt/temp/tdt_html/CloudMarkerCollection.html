CloudMarkerCollection

可以实现同时在地图上展示万级、密集的点数据。兼容chrome、safari、IE9及以上浏览器。

构造函数

构造函数 说明
CloudMarkerCollection(lnglats:Array<LngLat>,
opts:CloudMarkerCollectionOptions)		 创建海量点类。lnglats为点的坐标集合，opts为点的绘制样式。

方法

方法 返回值 说明
setLnglats (lnglats:Array< LngLat >) none 设置要在地图上展示的点坐标集合。
setStyles(styles:CloudMarkerCollectionOptions) none 点的样式，包括:大小"size"（可选,默认正常尺寸10*10px，SizeType类型），形状"shape"（可选，默认圆形，ShapeType类型），颜色"color"（可选，字符串类型）。
clear() none 清除海量点。

事件

事件 参数 描述
click {type,target,lnglat,containerPoint} 海量点的点击事件
mouseover {type,target,lnglat,containerPoint} 海量点的移入事件
mouseout {type,target,lnglat,containerPoint} 海量点的移出事件
