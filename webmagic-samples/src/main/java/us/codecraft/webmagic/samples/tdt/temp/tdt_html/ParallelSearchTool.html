ParallelSearchTool

构建一个绘制平行搜寻区的工具类。

构造函数

构造函数 说明
ParallelSearchTool(map:Map,options:ParallelSearchToolOptions) 构造一个绘制平行搜寻区的工具。

方法

方法 返回值 说明
clear() none 清空工具绘制所有平行搜寻区。
getLayers() Array<ParallelSearch> 获取所有编辑完成的平行搜寻区。
open() boolean 开启平行搜寻区工具。
close() none 关闭平行搜寻区工具。
addEventListener(event:String, handler:Function) none 添加事件监听函数。
removeEventListener(event:String, handler:Function) none 移除事件监听函数。

事件

事件 参数 说明
click {type,target,currentLnglats,currentLayer,allLayers} 工具点击一次地图触发的事件。
move {type,target,currentLnglats,currentLayer,allLayers} 工具在绘制图形时移动鼠标的事件。
dbclick {type,target,currentLnglats,currentLayer,allLayers} 工具双击结束编辑时触发的事件。
