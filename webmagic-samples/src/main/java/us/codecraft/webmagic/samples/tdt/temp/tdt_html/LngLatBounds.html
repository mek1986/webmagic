LngLatBounds

此类可以指定的坐标范围建立一个矩形区域。

构造函数

构造函数 说明
LngLatBounds(sw:LngLat,ne:LngLat) 根据给定的多个地理位置坐标点创建矩形。
参数说明：
sw：矩形区域的西南角。
ne：矩形区域的东北角。

方法

方法 返回值 说明
getSouthWest() LngLat 返回西南点坐标。
getNorthEast() LngLat 返回东北点坐标。
getCenter() LngLat 返回边界的中心点。
extend(lnglat:LngLat) this 放大此矩形，使其包含给定的点。
参数说明：
lnglat：点的地理坐标。
contains(other:LngLat|LngLatBounds) boolean 如果点的地理坐标位于此矩形内或者传入的矩形区域完全包含于此矩形区域中，则返回true。
参数说明：
other：点的地理坐标或者地理范围坐标。
intersects(other:LngLatBounds) boolean 如果矩形与给定边界相交则返回true。
参数说明：
other：地理范围坐标。
