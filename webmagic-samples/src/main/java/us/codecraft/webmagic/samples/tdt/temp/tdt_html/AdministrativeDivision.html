AdministrativeDivision

此类用于获取行政区划信息。

构造函数

构造函数 说明
AdministrativeDivision () 创建一个获取行政区划的实例。

方法

方法 返回值 说明
search(config: AdministrativeDivisionOptions, callback: Function) none 根据检索词发起检索。
参数说明：
config：详见AdministrativeDivisionOptions类。
callback：设置回调函数。回调函数参数为AdministrativeDivisionResult，具体内容详见AdministrativeDivisionResult类。
