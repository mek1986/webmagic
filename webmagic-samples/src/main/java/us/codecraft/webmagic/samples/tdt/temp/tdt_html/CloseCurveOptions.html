CloseCurveOptions

此类用来表示地图上构造一个闭合曲线的覆盖物类。

属性

属性 类型 默认值 说明
color string "#0000FF" 闭合曲线的线颜色。
weight number 3 闭合曲线的宽度，以像素为单位。
opacity number 0.5 闭合曲线的透明度（范围0-1 之间）。
lineStyle string "solid" 闭合曲线的线样式（solid或dashed）。
fillColor String "#0000FF" 闭合曲线填充颜色。当参数为空时，覆盖物将没有填充效果。
fillOpacity number 0.2 闭合曲线填充的透明度（范围0-1 之间）。
